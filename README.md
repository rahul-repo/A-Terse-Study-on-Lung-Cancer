# Lung Cancer Study (2024)

## Overview

This project is a group-based epidemiological study analyzing lung cancer patterns using global and Indian datasets. The objective is to examine distributional and demographic characteristics of lung cancer and understand regional variation in incidence patterns.

The study is conducted within a public health and statistical analysis framework.

---

## Project Year: 2024

## Objectives

* Analyze lung cancer incidence patterns across datasets
* Study distribution across cancer types (SCLC, NSCLC, others)
* Examine TNM staging structure
* Analyze gender-wise and regional variation
* Identify key risk factors associated with observed trends

---

## Data

The study uses epidemiological datasets containing:

* Lung cancer incidence records
* Cancer type classification (SCLC, NSCLC, others)
* TNM staging information
* Gender-wise distribution
* Regional and global cancer statistics
